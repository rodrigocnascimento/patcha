## Requirements

- Node.js version 18 or higher
- npm, pnpm, or yarn

## Install Globally

```bash
npm install -g patcha
```

## Use with npx

Run without installing:

```bash
npx patcha scan
```

## Verify Installation

```bash
patcha --version
```

## Update Patcha

```bash
npm update -g patcha
```

## Uninstall

```bash
npm uninstall -g patcha
```
